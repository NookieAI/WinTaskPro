# WinTaskPro

A modern, dark-themed manager for the Windows Task Scheduler — view, create, edit,
run and monitor every scheduled task (including hidden system tasks) from one clean UI.

> **This repository hosts the release binaries only.**
> The source code is maintained privately; each release here is built and published
> automatically by CI from the private source repo.

## Download

Grab the latest **`WinTaskPro.exe`** from the **[Releases page](https://github.com/NookieAI/WinTaskPro/releases/latest)**.
It is a single-file portable executable — run it from anywhere, no installer.

Direct link (always the newest build):

```
https://github.com/NookieAI/WinTaskPro/releases/latest/download/WinTaskPro.exe
```

## Requirements

- Windows 10 21H1 or later (64-bit)
- WebView2 Runtime (pre-installed on Windows 10 21H1+ and Windows 11)
- Run as Administrator (required for Task Scheduler access)

## Auto-update

Installs from 1.8.0+ check this repository on launch and can self-update in place
when a new release is published — click **Update Now** in the in-app banner.
